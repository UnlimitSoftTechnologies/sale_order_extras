Sales Order Line Profit Margin

Manage profit margins and base prices directly on sales order lines, giving your sales team greater control over pricing strategies.

Fully compatible with Odoo Community and Enterprise (Odoo SH and on-premise deployments).

Features

  • Enable profit margin management from the Sales module settings.
  • Automatically assigns authorized users to the Use Profit Margin security group.
  • Allows users in this group to define a profit margin and a base price per sales order line.
  • Dynamic recalculation between base price, profit margin, and unit price.
  • Works seamlessly with standard Odoo Sales workflows.

How It Works

After installing the module, activate the profit margin option from the Sales settings. Once enabled, users assigned to the Use Profit Margin group will be able to define a base price and a profit margin on each sales order line. The system automatically recalculates the unit price based on the values entered, allowing users to simulate different pricing scenarios by adjusting the profit margin, base price, or unit price as needed.
