PoS Session Logout Email Notification

Automatically send email notifications when PoS sessions are logged out or closed, keeping your team informed of session summaries and transaction details.

Compatible with Odoo SH and on-premise deployments. A configured outgoing mail server is required for sending notifications.

Features

  • Automatic email notifications when PoS sessions are logged out or closed.
  • Includes session details and transaction summaries in the notification.
  • Improves oversight and monitoring of your Point of Sale operations.
  • Works with standard Odoo PoS.

How It Works

Once the module is installed, you can enable the email notification option in the PoS settings. After a PoS session is closed or logged out, the system will send a summary email with key session figures and transaction details to the selected list of users. Ensure that your instance has a correctly configured outgoing mail server to deliver the notifications. Odoo SH setups already include a mail server for basic emails; for on-premise instances, an external outgoing server must be configured for email delivery.
